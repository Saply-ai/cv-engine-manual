User Manual - CV Engine

Overview The CV Engine allows users to format CVs into predefined templates by simply sending an email with the CV(s) as an attachment. The tool automatically processes the CV(s) and returns the formatted output.

How to Use the Tool

  1. Basic Usage

    • Send an email to engine@saply.ai with the CV(s) attached.
    • If no specific instructions are provided, the CV will be formatted using the default template.

  2. Using Custom Templates

    • Specify the name of a desired template in the email subject to format the CV accordingly.
    • If multiple CVs are attached, each will be formatted using the specified template.

  3. Multiple CV Processing

    • The tool supports batch processing. If multiple CVs are attached, each one will be formatted separately into the requested template.

  4. Including Vacancy Matching

    • If a vacancy description is included in the email body (not as an attachment), the tool will analyse the CV(s) and provide additional insights about their compatibility with the vacancy.

  5. CV Tailoring (Pimp Mode)

    • If "pimp" is added to the email subject, the CV will be tailored to the vacancy description provided in the email body.
    • This includes rewriting the introduction and reordering the skills list to better match the vacancy.

Token Consumption & Processing Rules

  • A token is consumed for each successful CV output.
  • If formatting fails and no result is produced, no token is deducted.

Example Scenarios

  1. Format a CV using the default template
    • Send an email with a CV attached and no subject.
  2. Format a CV using a specific template
    • Send an email with "bnp" in the subject and a CV attached.
  3. Format multiple CVs with a specific template
    • Send an email with "bnp" in the subject and multiple CVs attached.
  4. Match CV(s) with a vacancy description
    • Send an email with a CV attached and a vacancy description in the email body.
  5. Tailor CV(s) to a vacancy
    • Send an email with "pimp" in the subject, a CV attached, and a vacancy description in the email body.
