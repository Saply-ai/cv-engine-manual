CV Engine — User Manual

CV Engine allows you to easily format, translate, and tailor CVs into your chosen template. Simply send an email with your CV(s) attached and let the engine do the rest!

How to Use

  1. Send Your CV(s)

    • Email your CV(s) as attachment(s) to engine@saply.ai.
    • If you donâ€™t include a subject, the default template and original CV language are used.

  2. Specify a Template (optional)

    • Add the template name (e.g., bnp) in the email subject to use a specific template.
    • Each attached CV will be formatted using this template.

  3. Request Translation (optional)

    • Add french, dutch, english, or german in the email subject to receive the formatted CV in that language.
    • You can combine template and language: bnp dutch
    • If no language is specified, the CV remains in the language of the original file.
    Note: Order does not matter in the subject (e.g., dutch bnp also works).

  4. Batch Processing

    • You can attach multiple CVs. All CVs will be processed with the same options specified in the email subject and body.
      For example, if your subject is bnp pimp french and you paste a vacancy description in the email body, all attached CVs will:
      • Use the BNP template,
      • Be tailored ("pimped") to the same vacancy,
      • Be translated into French.

  5. Vacancy Matching (optional)

    • Paste a vacancy description into the email body (not as an attachment).
    • The engine will analyze and match the CV(s) to the vacancy, adding compatibility insights.

  6. Tailor CV to Vacancy ("Pimp Mode") (optional)

    • Add pimp in the subject, attach a CV, and include the vacancy description in the email body.
    • The introduction and skills will be rewritten and reordered to fit the vacancy requirements.

Token Usage & Processing Rules

  • One token is used for each successful CV output.
  • No token is used if processing fails and no result is produced.

Examples

Format a CV using the default template:
  • Send an email with a CV attached and no subject.
Format a CV using a specific template:
  • Subject: bnp
    Attach CV(s).
Format a CV in a specific language:
  • Subject: dutch (for Dutch output)
  • Or combine: bnp german (use template BNP and translate to German)
  • Attach CV(s).
Batch process multiple CVs:
  • Subject: bnp french
  • Attach several CVs.
Match CV(s) to a vacancy description:
  • Attach CV(s), paste vacancy description in the email body.
Tailor CV(s) to a vacancy:
  • Subject: pimp (or pimp dutch, etc.)
  • Attach CV(s), paste vacancy in the email body.
